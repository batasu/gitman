# Git Manager 
A basic git manager to auto generate 

* Basic options 
  - status
    > to view `git status`
  - auto add
    > auto add files on git 
  - auto commit
    > auto commit `auto commit - updated`
  - pull
  - push
  - log
    > to view log in one line 
  - quit
